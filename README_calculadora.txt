
README - Calculadora Interativa em Python

**Bem-vindo(a)!**
Este é um projeto simples, mas poderoso: uma calculadora interativa feita em Python, pensada para quem quer praticidade na hora de resolver contas rápidas ou funções matemáticas um pouco mais avançadas.


**O que dá pra fazer com ela?**
Você escolhe a operação e o programa faz o resto:

 - Soma

 - Subtração

 - Multiplicação

 - Divisão

 - Fatorial (limitado a 10 para evitar resultados enormes)

 - Porcentagem

 - Raiz quadrada

 - Logaritmo (qualquer base, inclusive o log natural)

 - Seno, cosseno e tangente de ângulos


**Como usar?**

 1) Execute o arquivo .py no terminal ou no Google Colab.

 2) Digite seu nome — um toque pessoal sempre é bem-vindo.

 3) Escolha o número da operação desejada no menu.

 4) Preencha os valores pedidos.

 5) Veja o resultado aparecer na tela!

 6) No final, você escolhe se quer continuar ou encerrar.


**Extras Legais:**

Nome com estilo: O nome digitado aparece com a primeira letra em maiúscula.

Tela limpa: A cada nova operação, a tela é limpa, dependendo do sistema (Windows, Linux ou Colab).

Tratamento de erros: O código evita travamentos caso você digite algo errado.

Fatorial controlado: Se o número for muito grande, o resultado aparece em notação científica.
